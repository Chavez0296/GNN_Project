# Stage 1 - Subset 4 (Baseline Diagnostics)

This subset performs diagnostics requested in the project PDF after baseline establishment.

## File

- `stage1_subset4_diagnostics.py`

## What it does

1. Relation frequency slices
   - Computes relation frequencies from train edges.
   - Splits relations into rare/medium/frequent buckets.
   - Aggregates per-relation MRR from available baseline output files.

2. Degree slices
   - Builds train-graph entity degree statistics.
   - Computes degree-sliced MRR (low/medium/high) on sampled test queries.

3. Error case mining
   - Extracts high-confidence false positives and false negatives.
   - Assigns lightweight failure labels (hub bias, rare relation, no supporting path, cold start, ambiguous signal).

## Inputs

Defaults are configured for the outputs generated by Subset 1/2/3:

- `outputs/stage1_subset1_metrics.json`
- `outputs/stage1_subset2_distmult_metrics.json`
- `outputs/stage1_subset2_complex_metrics.json`
- `outputs/stage1_subset3_rgcn_metrics.json` (optional)

## Command

```bash
python stage1_subset4_diagnostics.py
```

Example with faster diagnostic sample:

```bash
python stage1_subset4_diagnostics.py --max_test_samples 1500 --template_examples 300 --template_topk 12
```

## Outputs

- `outputs/stage1_subset4_diagnostics.json`
- `outputs/stage1_subset4_diagnostics.md`

The JSON output contains structured slices and error records. The Markdown file is a quick readable summary.
